Build a new blog for free with free SSL certificate.

This blog was basically built with Github page, but it has some problems.
Despite you can build your static website in few minutes using Github page, it lacks functionality such as TSL/SSL communication and force SSL for that. I am using netlify instead to solve those problems.

I’m going to explain how you build free SSL blog here.

What you will use are:
- Hugo
- Github
- Netlify

Hugo is one of framework that you can build your article and generates static pages on your local environment. This framework is made by Golang, which I love the most.
It has so many templates and you can easily customize. Why not choosing this framework! You have other choices if you do not like it. Octopress, Hexo and Pelican are also static web site generator, and build your own website.

To get started, see this documentation and follow them.

IMPORTANT: The configuration is bit tricky. Need to see examples what you can set for your website. It depends on templates you chose. For example see this page if you chose hugo-initio template.

Netlify allows you to host your web site, with SSL certificate. As you see this issue, you are not able to add your SSL certificate to make your site HTTPS with your custom domain.
Netlify helps you for it.

The easiest way to build your application is to configure with your Github account and setup.
Build your app, configure your DNS domain and setup SSL, and you can set force SSL which force user to use HTTPS access. You need at least one hour to enable SSL with letencrypt certificate.

Now, this blog will generate and show new articles as soon as I push to master branch, Netlify will build and publish by your source code.

Conclusion

In this way I show you, knowledge of Git and Markdown are required, however, I’m sure this is the cheapest way and easy to setup SSL, including force SSL.
